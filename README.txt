Application with laravel php framework, mysql  and Bootstrap css.
CRUD with admin area using vue.
Eshop with Paypal payment.

-------------------

Require: 
Composer, laravel,
apache2, mysql,
PHP >= 7.2.5 see extensions requirements https://laravel.com/docs/7.x/installation
( WAMP or LAMP or MAMP confiqured installation),
Vue CLI4,
Paypal sandbox account.
